1. styled-components
- props 형태로 스타일 정의 가능
- 기존에 정의해놓은 스타일 컴포넌트의 값을 가져다가 새로운 스타일 컴포넌트를 생성
- 처음 정의한 스타일컴포넌트의 태그 값을 원하는데로 수정할 수 있음
- nesting 기능 / 애니메이션 기능
- 스타일의 템플릿화가 가능하다
- 스타일 컴포넌트는 객체 자료형태의 theme(*다크모드 / 라이트모드)
> theme 정의하는 장소는 APP컴포넌트의 상위경로
> useContent / Redux => Root 컴포넌트 위에다가 설정 => Root 컴포넌트 하위 존재 컴포넌트들이 data값을 받을 수 있었음
